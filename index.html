HTML

  • HTML serves as the frame for building a webpage.
  • The head element contains information about the webpage.
  • This information is called metadata and isn't visible to the user of a webpage.
  • It includes things like charset, the compatability, and the title (not header).
  • The body element represents the visible content shown to the user.
  • The body contains information that the user can see like the headers, bulletpoints, etc.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To incorporate CSS into your website you can choose inline, internal, and external formatting.
  • * acts as a wildcard to declare ALL elements on a page have the following style properties assigned to them

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Comments in JS written like /* code */ or for short ones //
  • Conditionals are code structures used to test if an expression returns true or not/
  • Functions are a way of packaging functionality you want to reuse
  • For loops use the predictable pattern of indices to perform a task on all items in an array by allowing a single code block to be executed over and over
  • Three statements in a for loop are 1)starting point 2) condition 3) What allows the array to iterate over each item
